先记住三句话

  1. 先看口径，再看数据。
  2. 先看时间，再下结论。
  3. 热度不等于收益，必须联合判断。

时间怎么看（最重要）

  • 最新数据日期：数据属于哪一天。
  • 本次爬取时间：系统抓取这批数据的时间。
  • 源数据更新时间：原网页接口返回批次（常见 11/14/15/16/21 点）。
汇报时建议固定说法：截至【日期】，源数据更新时间【时间】，在【范围/视角】下结论如下。

推荐使用顺序

  1. 核心看板：先看当日与滚动排序。
  2. 动态指标：判断是短期热点还是趋势延续。
  3. 竞品弱点：找可突破标的。
  4. 详情驾驶舱：验证用户行为与互动强度。

各页重点

  • 核心看板：加仓/减仓/双边/当日排序；当日排序口径固定“全部基金”。
  • 动态指标：支持搜索基金名称/代码/公司；看净偏好、热度、分歧预警。
  • 竞品弱点：按弱点分排序挑重点观察对象。
  • 详情驾驶舱：支持搜索基金名称/代码/公司；看浏览-关注-持有与内容互动。
  • 指标文档：口径词典与指标解释，汇报优先引用这里。

常见问题

  • 同一基金在不同页不一致：先核对“当前口径提示”（日期/范围/视角/检索词）。
  • 搜不到“宝盈”：先确认日期与范围；当日样本池可能无该基金。
  • 同一天数字变化：可能是源站批次更新，先对照“源数据更新时间”。
  • 页面看起来没更新：先强刷浏览器（macOS：Cmd + Shift + R）。
